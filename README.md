# Stickman Geometry Lab 🎓 -

A hands-on, interactive educational game built with Python and Pygame-CE. Designed to help a 5-year-old learn shapes, angles, and colors through physics-based gameplay and drag-and-drop interaction.

# 🚀 Features -
Physics-Driven Movement: A stick-man character with gravity, jumping, and collision detection for multiple geometric shapes.
Dynamic Slopes: Custom math-based collision for a triangle "slide."
Radial Collision: Pythagorean-based physics for landing on circular objects.
Interactive Learning: A drag-and-drop "Shape Quiz" system where labels can be moved around the screen.
Cyberpunk Visuals: High-contrast RGB colors and a custom 1920x1080 Full HD environment.

# 🛠️ Tech Stack -
Language: Python 3.14
Library: Pygame-CE (Community Edition)
Math: Trigonometry and Vector physics for character movement.
